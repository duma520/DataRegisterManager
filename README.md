## 第一部分：欢迎与开篇

### 1.1 软件简介：它是什么？
《资料登记管理系统》是一款**免费、开源、跨平台**的桌面应用程序。简单来说，它就像是一个**加强版的、私有的、多用户的电子表格**。您可以用它来记录任何结构化的信息，例如：客户资料、产品清单、个人账目、学习笔记、工作任务、库存盘点……几乎任何您能想到的，需要以表格形式记录和管理的内容，它都能胜任。

### 1.2 目标用户：谁可以用它？
*   **家庭用户：** 记录家庭收支、亲朋好友联系方式、食谱收藏、旅行计划。
*   **学生：** 整理课程表、记录实验数据、管理学习资料、制作错题本。
*   **个体经营者：** 管理客户信息、记录进货/出货流水、统计销售数据。
*   **公司职员：** 作为项目任务看板、部门工作计划表、数据收集工具。
*   **IT/专业人员：** 可以作为轻量级的数据库前端，用于数据采集和原型验证。

### 1.3 核心价值：它能帮你做什么？
1.  **数据集中化：** 所有信息存放在一个数据库中，避免文件散落各处。
2.  **多用户隔离：** 不同用户可以登录自己的账户，查看和编辑自己的数据，互不干扰。
3.  **灵活的结构：** 您可以自由定义“单元格”（独立表格）的数量，以及每个“单元格”内部的行列数，完美适应各种业务场景。
4.  **安全可靠：** 内置的自动备份和手动备份功能，确保您的数据万无一失。
5.  **高效检索：** 强大的搜索功能，让您在海量数据中快速找到所需信息。

## 第二部分：零基础入门篇 (像用Excel一样简单)

这一部分是写给**完全不懂技术**的普通用户看的，我们将用最简单的语言和操作，带您迈出第一步。

### 2.1 下载与启动
1.  **获取程序：** 访问项目的GitHub发布页面，下载适用于您操作系统的最新版本（例如 `DataRegisterManager.exe` 文件）。
2.  **启动软件：** 双击下载的文件，程序就会启动。您会看到一个蓝色的图标和程序启动的打印信息（这些信息是后台运行的，不用理会）。

### 2.2 登录系统
程序启动后，第一个出现的窗口就是**登录界面**。
1.  **用户名：** 点击下拉框，您会看到系统已经为您创建了一个管理员用户 `admin`。您可以直接选择它。
2.  **密码：** 管理员用户的默认密码是 **`admin123`**。请在密码框中输入它。
3.  **点击“登录”按钮**，您就成功进入主界面了。

> **💡 小提示：** 您也可以点击“用户管理”按钮，为自己创建一个新账户。这部分我们将在进阶篇详细讲解。

### 2.3 第一次使用：认识主界面
登录后，您会看到一个宽敞的窗口，主要分为以下几个部分：
*   **顶部菜单栏：** 包含了“文件”、“用户”、“单元格”、“视图”、“帮助”等所有功能的入口。
*   **顶部工具栏：** 一些常用功能的快捷按钮，比如“备份管理”、“刷新”、“设置列标题”和“搜索框”。
*   **用户信息栏：** 显示当前登录的用户名和您的登录时间。
*   **单元格选择器：** 这是一个下拉框，默认有“单元格 1”、“单元格 2”……“单元格 6”。点击它们可以切换不同的表格。
*   **主工作区：** 一个带有标签页的区域，目前显示的是“单元格 1”的表格。这个表格默认有 **3行4列**，就像一个迷你Excel表。
*   **底部状态栏：** 显示“就绪”字样和进度条，用于提示系统状态。

### 2.4 快速上手：我的第一个“单元格”
想象一下，您想用这个系统来记录您的**家庭购物清单**。
1.  在**单元格选择器**中，确保当前选中的是“单元格 1”。
2.  您看到的这个3行4列的表格，就是您的“购物清单”草稿本。
3.  但是3行可能不够用？别担心，我们马上教你如何调整。

### 2.5 录入数据：像填表一样简单
1.  **直接点击表格中的格子**，比如第一行第一列。
2.  输入“苹果”，按键盘上的 `Tab` 键或 `→` 键，光标就会自动移动到右边一格。
3.  接着输入“5元”，然后继续输入“香蕉”、“3元”……
4.  您会发现，输入完一行后，可以按 `↓` 键继续下一行。

**举几个不同行业的例子：**
*   **教师：** 可以用“单元格 1”记录“学生成绩表”（列：姓名、语文、数学、英语），用“单元格 2”记录“考勤表”。
*   **财务：** 可以用“单元格 1”记录“日常支出”（列：日期、类别、金额、备注），用“单元格 2”记录“收入明细”。
*   **个人学习：** 可以用“单元格 1”记录“读书笔记”（列：书名、作者、金句、感悟）。

### 2.6 保存与搜索：再也不怕找不到记录
*   **自动保存：** 当您**离开当前编辑的格子**（点击其他格子或按Tab键）时，系统会**自动保存**您输入的内容。您完全不需要手动点击“保存”按钮，非常方便！
*   **搜索数据：** 如果您在购物清单里记录了很多东西，想找到“苹果”的信息，怎么办？
    1.  在顶部的**工具栏**中，找到“搜索”框。
    2.  输入“苹果”，然后按回车或等待片刻。
    3.  系统会弹出一个对话框，告诉您在哪里找到了“苹果”，以及它旁边的“价格”是多少。

### 2.7 切换账户与退出
*   **切换用户：** 点击顶部菜单栏的 **“用户” -> “切换用户”**，系统会询问您是否确认。点击“是”后，就会回到登录界面，您可以换一个账户登录。
*   **退出程序：** 点击右上角的“X”关闭按钮，或点击菜单栏的 **“文件” -> “退出”**。

## 第三部分：进阶应用篇 (让工作更智能)

这一部分适合已经学会基础操作，想要根据自己的需要调整软件的用户。

### 3.1 用户管理：给家人/同事创建账户
1.  在登录界面，点击 **“用户管理”** 按钮。
2.  在弹出的对话框中，点击 **“添加用户”**。
3.  输入新用户的**用户名**（例如：`zhangsan`）和**密码**（可以为空）。点击“确定”。
4.  现在，您可以在登录界面选择 `zhangsan` 这个用户登录了。新用户的数据与 `admin` 完全隔离，非常安全。

### 3.2 自定义“单元格”数量：增加或减少你的工作台
如果您觉得6个单元格（表格）不够用，或者太多了，可以随时调整。
1.  点击菜单栏 **“用户” -> “编辑配置”**。
2.  在弹出的“用户配置”对话框中，找到 **“单元格数量配置”** 区域。
3.  调整 **“单元格数量”** 后面的数字。比如，改成 `10`，表示您将有10个独立的表格。
4.  点击“确定”。系统会刷新界面，您现在可以从下拉框或标签页中选择使用1-10号单元格了。
> **专业应用举例：** 一个项目经理可以为每个项目建立一个“单元格”（如“A项目”、“B项目”），每个项目下面再建立任务清单（行）和任务状态（列）。

### 3.3 改变表格大小：调整行数与列数
您可以为**每一个**单元格（表格）单独设置它有多少行多少列。比如，您的“购物清单”需要10行，而“工作计划”只需要5行。
1.  在界面中，先通过**下拉框或标签页**切换到您想修改的那个单元格（比如“单元格 1”）。
2.  点击菜单栏 **“单元格” -> “编辑当前单元格配置”**。
3.  在弹出的对话框中，修改 **“行数”** 和 **“列数”**。比如，将行数改成 `10`。
4.  点击“确定”。您会看到“单元格 1”立即变成了10行4列。
5.  系统会询问您“是否将此单元格配置设为新建单元格的默认配置？”，选择“是”的话，以后新建的单元格都会采用这个配置；选择“否”则只修改当前这一个。

### 3.4 给表格起名字：设置列标题
默认情况下，表格的列是没有标题的。为了让数据更清晰，您可以给每一列起一个名字。

#### 3.4.1 统一标题模式
这种模式下，所有列的标题都相同。
1.  进入“用户配置”对话框（用户->编辑配置）。
2.  在 **“单元格默认配置”** 区域，勾选 **“需要标题”**。
3.  在 **“标题模式”** 下拉框中选择 **“统一标题”**。
4.  在 **“统一标题文本”** 框中输入您想要的标题，例如“购物项目”。
5.  点击“确定”。您会发现当前所有单元格的所有列，标题都变成了“购物项目”。

#### 3.4.2 独立标题模式
这种模式下，您可以给每一列设置不同的名字，例如“名称”、“单价”、“数量”、“备注”。
1.  切换到您想要修改的单元格（比如“单元格 1”）。
2.  点击工具栏上的 **“设置列标题”** 按钮，或通过菜单栏 **“单元格” -> “设置当前单元格列标题”**。
3.  在弹出的对话框中，您可以看到每一列都有一个输入框。
4.  分别输入您想要的标题，例如“商品名称”、“价格”、“数量”、“备注”。
5.  点击“确定”。您会发现该表格的每一列现在都有了各自独立的标题。

### 3.5 自动调整列宽：让表格更整洁
为了让您输入的内容能完整显示，系统默认开启了自动调整列宽功能。
1.  如果您在“用户配置”中看到了 **“根据填充内容自动调整列宽”** 的复选框，勾选它即可启用。
2.  启用后，当您输入的内容过长时，该列的宽度会自动变宽，让所有内容都显示出来。如果您不想要这个功能，可以取消勾选。

## 第四部分：高级应用与专业篇 (像专家一样掌控)

这一部分是写给深度用户、技术爱好者和专业管理人员的，旨在让您彻底挖掘这个软件的潜力。

### 4.1 深入理解“单元格”：项目、部门、个人的数据容器
在专业场景下，“单元格”可以被视为一个**完全独立的、可配置的数据容器**。它不仅仅是一个表格，而是一个可以承载特定业务逻辑的数据单元。例如：
*   **在IT项目中**，单元格1可以是“需求列表”（列：需求ID、描述、优先级、状态），单元格2可以是“Bug列表”，单元格3可以是“迭代计划”。
*   **在客户关系管理（CRM）中**，单元格1可以是“客户基本信息”，单元格2可以是“跟进记录”，单元格3可以是“销售机会”。
*   **在库存管理中**，单元格1可以是“入库记录”，单元格2可以是“出库记录”，单元格3可以是“库存盘点表”。

### 4.2 每个“单元格”的独立配置
这正是本系统的强大之处。您可以为每个“单元格”配置完全不同的行列数、标题模式和功能选项。这通过“编辑当前单元格配置”实现。
*   **配置隔离：** 修改“单元格 1”的行数，不会影响“单元格 2”的列数。
*   **配置继承：** 当您在编辑某个单元格的配置时，系统会询问是否将其设为“默认配置”。设为默认后，**所有新创建的单元格**都会自动采用这套配置，极大地提高了配置效率。

### 4.3 登录时间与相隔天数：追踪记录时效性
*   **记录登录时间：** 当用户登录时，系统会记录下精确的登录时间。每条数据在保存时，都会关联这个登录时间。
*   **计算相隔天数：** 如果在“单元格配置”中开启了“计算相隔天数”，那么在后续版本中（或基于此数据），您可以方便地知道某条记录是登录后第几天录入的。这对于需要监控记录时效性的场景非常有用，例如“反馈处理时限”、“任务完成周期”等。

### 4.4 备份与恢复：为数据上“双保险”
数据是无价的，备份功能是系统的核心安全组件。

#### 4.4.1 手动备份：重要操作前必做
在您进行大规模修改、删除或配置变更前，建议先手动备份。
1.  点击菜单栏 **“文件” -> “备份管理”**，或工具栏上的 **“备份管理”** 按钮。
2.  在弹出的“数据库备份恢复”窗口中，点击 **“手动备份”** 按钮。
3.  系统会立即创建一个以当前时间为名的备份文件（如 `backup_20260326_143022_manual.db`），并存放在程序目录下的 `backups` 文件夹中。

#### 4.4.2 自动备份：系统的定时守护者
系统在后台默默为您服务。只要程序在运行，它就会**每隔一小时**自动创建一次备份（备份类型为 `auto`）。您无需任何操作，这为您提供了额外的保障，即使您忘记手动备份，也能找回最近一小时的数据。

#### 4.4.3 从备份恢复：时光倒流，找回数据
如果不小心删除了重要数据，或者配置改错了，可以轻松恢复。
1.  打开“备份管理”窗口。
2.  在备份列表中，您可以通过筛选（类型、时间）找到您想要恢复的那个备份。
3.  **双击**该备份，或在列表中找到它，点击右侧的 **“恢复”** 按钮，或选中它后点击下方的 **“恢复选中备份”**。
4.  系统会弹出确认框，并告知您将创建回滚备份。点击“确定”。
5.  恢复完成后，当前数据库会被替换成备份时的状态，同时系统会为您创建一个名为 `rollback_..._before_restore.db` 的回滚备份，以防万一。界面数据会自动刷新。
> **专业提示：** 备份文件是标准的SQLite数据库文件。高级用户可以直接使用数据库工具（如DB Browser for SQLite）打开它们进行查看和分析。

### 4.5 调试模式：透视系统后台运行
对于技术专家或希望深入了解软件运行情况的用户，可以开启“调试模式”。
1.  点击菜单栏 **“视图” -> “调试模式”**。
2.  开启后，底部状态栏的文字会变成红色，并且当您每次保存数据时，状态栏会显示“数据已保存 - 单元格X (Y, Z)”的信息，让您知道数据已成功写入数据库。

## 第五部分：常见问题与疑难解答 (FAQ)

### 5.1 安装与启动问题
*   **Q: 运行程序时提示缺少某某.dll文件怎么办？**
    A: 这通常是Python环境或打包问题。请确保您下载的是完整版本，或者尝试以管理员身份运行。如问题持续，请在GitHub Issues中提供完整的错误截图。
*   **Q: 程序无法启动，闪退怎么办？**
    A: 请检查您的操作系统版本，确保是Windows 7 SP1或更高版本。可以尝试在命令行中运行程序，查看输出的错误信息，这通常能帮助定位问题。

### 5.2 数据录入与保存问题
*   **Q: 我输入的数据，切换到别的单元格再回来，怎么不见了？**
    A: 请确认您是在**离开当前格子**时才触发的保存。如果您一直在编辑同一个格子，数据尚未保存。请尝试点击其他格子，再切换回来查看。
*   **Q: 我输入的内容很长，列宽没有自动调整怎么办？**
    A: 请检查“用户配置”中的“根据填充内容自动调整列宽”选项是否已勾选。如果已勾选，双击该列的分割线也可以手动调整。

### 5.3 备份与恢复问题
*   **Q: 备份文件占用了很多空间，可以删除吗？**
    A: 可以。系统会自动保留最近30个备份。如果您想手动删除旧备份，可以直接进入程序目录下的 `backups` 文件夹进行删除。请务必小心操作。
*   **Q: 我恢复了一个备份，但是想回到恢复前的状态怎么办？**
    A: 不用担心，每次恢复操作都会自动创建一个名为 `rollback_..._before_restore.db` 的回滚备份。您只需再次进入“备份管理”，找到并恢复这个回滚备份即可。

### 5.4 自定义配置问题
*   **Q: 为什么我修改了单元格的行数，但之前录入的数据不见了？**
    A: 数据没有丢失。当您减少行数时，被缩减掉的行中的数据仍然保存在数据库中，只是不再显示在界面上。如果您想找回，只需再次增加行数到大于之前数据的行号，数据就会重新出现。
*   **Q: 独立标题模式设置后，如何快速清空所有标题？**
    A: 在“列标题设置”对话框中，点击“统一设置”按钮，输入一个空文本，然后点击确定，即可将所有列的标题清空。然后您可以再切换回独立标题模式进行单独设置。
